# EchoPulse
EchoPulse is a machine learning pipeline for classifying lung and heart sounds using MFCC features with Random Forest and SVM models. It enables fast, audio-based detection of cardiopulmonary conditions through an ensemble prediction system.

Placed 2nd internationally out of 500+ teams at the HOSA International Leadership Conference (ILC) in Houston, Texas for developing EchoPulse after proposing the idea to real professionals, judges, and doctors from around the world.
